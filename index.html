Models Description
1

Face Detection

⬤ The model deals with "Face detection", It does so by identifying a face within an image and then proceeds to determine the number of faces present.
⬤ An error message is displayed if there is no face in image or the number of faces are greater than one.
⬤ If only one face is detected, the system proceeds to the next stage of the process, which includes detecting additional features such as eyeglasses, mask detection, orientation, and alignment, all within the predefined constraints.

2

EyeGlass Detection

⬤ The Gaussian filter is used in the detection of eyeglasses.
⬤ The Gaussian filter is used in the detection of eyeglasses. The filter detects the presence of edges around the glabella, which can indicate the presence of eyeglasses.
⬤ If the edge is present, the person is wearing glasses; if the edge is absent, the person is not wearing glasses.

3

Mask Detection

⬤ Mask detection is accomplished using Tensorflow model.
⬤ The face in the image is cropped and then feed to the model for mask detection in this process.
⬤ The model then determines whether or not user is wearing a mask in the image.

4

Background Color Detection

⬤ The background detection technique used in this module involves taking a sample of the background area and then averaging the pixel colour in the given area to get the mean background colour.

5

Face Orientation Detection

⬤ Face orientation detection involves first determining the position of the two eyes and then calculating the angle of the two eyes to the x-axis.
⬤ This angle can be used to determine the face's orientation.

6

Face Cervical Rotation Detection

⬤ In this module the face cervical rotation is checked, it is done by measuring the tangential angle between the distance between the two eyes and the distance from one eye to the nose.

7

Background Color changer

⬤ Using this model, background correction is done.
⬤ In this module, the gaussian filter is used with mediapipe to separate the background from edges.
⬤ Any colour can be used for the separated background.

8

Face Orientation Correction

⬤ This model is used to adjust the given user's face orientation.
⬤ The image is first converted to numpy, after which it is rotated using RotationMatrix2D, padding is then provided where necessary to the other portions.

9

Face Position Correction

⬤ The task of this model is face position correction.
⬤ If the face position in the image is not in the correct position.
⬤ Module first detects the face then crops it in a defined manner such that it can be replaced in the given image.

10

Image Dimension Correction

This module is applied when the image dimensions are not correct in the case of height and width of the image.
⬤ This model adds padding to the image vertically or horizontally without distorting the image and the face of the user.
